批量添加章节

说明
  1. 每行一个章节，并以第一个空格分割章节标识和章节标题
  2. 多个章节，请换行输入
  3. 每一行的格式，形如"ch01 第一章 Go语言从入门到精通"，这里，"ch01"则是章节标识，第一个空格以后的"第一章 Go语言从入门到精通"则是章节标题
  4. 如果章节标识已存在，则不会添加该章节
  5. 文档标识只能是数字、字母，以及“-”、“_”和“.”等字符，并且不能是纯数字
